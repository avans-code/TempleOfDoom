Design Pattern: Factory

Observer + Factory (10p)

Centralisatie van objectcreatie en DTO-mapping.


public class LevelFactory {
    public Level CreateLevel(LevelDto dto) {
        // Omzetten van JSON DTO naar rijke domein-objecten
        foreach (var roomDto in dto.Rooms) {
            var room = new Room(roomDto.Id, roomDto.Width, roomDto.Height);
            // ... logica voor enemies, items en deuren
            level.Rooms.Add(room.Id, room);
        }
        return level;
    }
}

Design Pattern: Observer

Observer + Factory (10p)

Event-driven communicatie voor Pressure Plates.


// PressurePlate (Subject)
public event Action? OnOccupancyChanged;
public bool IsOccupied {
    set { _isOccupied = value; OnOccupancyChanged?.Invoke(); }
}

// GameController (Observer)
private void WireObservers() {
    plate.OnOccupancyChanged += () => {
        if (plates.All(p => p.IsOccupied)) UnlockDoors();
    };
}

Design Pattern: Decorator

Decorator (10p)

Dynamisch gedrag toevoegen zonder klassen te wijzigen.


public class OpenOnOddDoor : IDoor {
    private readonly IDoor _innerDoor;
    
    public bool CanEnter(Player player, Room room) {
        // Eerst eigen regel (oneven levens), dan binnenste deur
        if (player.Lives % 2 == 0) return false;
        return _innerDoor.CanEnter(player, room);
    }
}

Design Pattern: Adapter

Adapter (10p)

Incompatibele DLL-interface koppelen aan eigen framework.


public class EnemyAdapter : Entity {
    private readonly CODE_TempleOfDoom_DownloadableContent.Enemy _dllEnemy;

    public void Move() {
        // Vertaal actie naar de DLL vijand
        _dllEnemy.Move();
        this.X = _dllEnemy.X;
        this.Y = _dllEnemy.Y;
    }
}
