The Factory Pattern

Creational Pattern
LevelFactory.cs 

public class LevelFactory {
    public Level CreateLevel(LevelDto dto) {
        Level level = new Level();
        foreach (var roomDto in dto.Rooms) {
            Room room = new Room(roomDto.Id, roomDto.Width, roomDto.Height);
            
            // Parse enemies using the Adapter (Polymorphism)
            ParseEnemies(room, roomDto.Enemies);
            
            level.Rooms.Add(room.Id, room);
        }
        return level;
    }
}

    set {

    }

The Decorator Pattern

Gedrag stapelen zonder inheritance hell.

Structural Pattern
OpenOnOddDoor.cs 

public class OpenOnOddDoor : IDoor {
    private readonly IDoor _innerDoor; // Decorated object

    public bool CanEnter(Player player, Room room) {
        // Decorator Logic (lives rule)
        if (player.Lives % 2 == 0) return false;
        
        // Delegate to original Door
        return _innerDoor.CanEnter(player, room);
    }
}

The Adapter Pattern

✅ Koppelt de onstabiele, incompatibele externe DLL_Enemy aan onze eigen Entity base class.

Structural Pattern

public class EnemyAdapter : Entity {
    private readonly External_DLL.Enemy _dllEnemy;

    public override void Move() {
        // Translate actions to DLL enemy
        _dllEnemy.UpdatePosition();
        this.X = _dllEnemy.CurrentX; // Sync position
    }
}
