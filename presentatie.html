Design Pattern: Factory

Creational
LevelFactory.cs 

public class LevelFactory {
    public Level CreateLevel(LevelDto dto) {
        Level level = new Level();
        foreach (var roomDto in dto.Rooms) {
            Room room = new Room(roomDto.Id, roomDto.Width, roomDto.Height);
            // Polymorfisme: Factory kiest de juiste implementatie
            ParseEnemies(room, roomDto.Enemies);
            level.Rooms.Add(room.Id, room);
        }
        return level;
    }
}
Factory slide: Vertel dat de CreateLevel methode de domme DTO's omzet naar de rijke domeinobjecten.

Design Pattern: Observer

Ontkoppelde, Event-Driven communicatie

Behavioral

// PressurePlate: De Publisher
public event Action? OnOccupancyChanged;
public bool IsOccupied {
    set { 
        if (_isOccupied != value) {
            _isOccupied = value; OnOccupancyChanged?.Invoke(); 
        }
    }
}
// GameController: De Subscriber
plate.OnOccupancyChanged += () => {
    if (plates.All(p => p.IsOccupied)) UnlockDoors();
};
Observer slide: Leg uit dat dit polling vervangt en hoe PressurePlate het Action event invoket.

Design Pattern: Decorator

Dynamisch gedrag stapelen zonder inheritance hell.

Structural

public class OpenOnOddDoor : IDoor {
    private readonly IDoor _innerDoor; // Wrapped object

    public bool CanEnter(Player player, Room room) {
        // Decorator logica
        if (player.Lives % 2 == 0) return false;
        
        // Delegeer naar de basis-deur
        return _innerDoor.CanEnter(player, room);
    }
}
Decorator slide: Laat zien hoe _innerDoor wordt ingepakt in de constructor om het gedrag dynamisch te stapelen.

Design Pattern: Adapter

Adapter voor externe, onstabiele DLL. We beschermen ons domein en coördinaten-systeem met de EnemyAdapter.


public class EnemyAdapter : Entity {
    private readonly CODE_TempleOfDoom_External_Enemy _dllEnemy;

    public override void Move() {
        _dllEnemy.UpdateDllPosition(); // Vertaling
        this.X = _dllEnemy.CurrentX; // Adapt coördinaten
    }
}

Polymorfisme & Clean Code

  • ✅ Geen Switch Statements: item.Interact() delegeert item-logica.
  • ✅ Information Expert: Room berekent zijn eigen enemies en collision logic.
  • ✅ Code Smells opgelost: 'God Class' is verwijderd, 'Inappropriate Intimacy' is opgelost via IDoor interface abstraction.
